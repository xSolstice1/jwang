## `> FEATURES — QUEST LOG`

### `⚔️ BOSS FIGHT: The Gatekeeper`

Side-scrolling mini-game loading screen. Slash through slimes, skeletons, and mages — then face the Gatekeeper boss to unlock the portfolio. Canvas-rendered pixel sprites, knockback physics, HP system.

```
    ╭─────╮       ╭───╮
    │ ◉ ◉ │  VS   │►◄ │  ← you
    │  ▬  │       │/▌\│
    ╰──┬──╯       ╰─┬─╯
      /|\          / \
```

### `🎯 MINI-GAME: Pipeline Runner`

Catch falling tech items (Python, AWS, Neo4j...) and dodge bugs. Combo multipliers, escalating difficulty, XP rewards.

### `📈 XP & LEVELING SYSTEM`

Every interaction earns XP. Scroll far enough — XP. Beat a game — XP. Solve a puzzle — XP. Level up and watch the progress bar fill.

### `>_ TERMINAL`

Hidden CLI overlay. Type `help` to start. Solve 4 coding puzzles (Two Sum, stack ops, Big O, linked lists) to unlock secrets like matrix rain, glitch mode, and the Konami code.

### `🕹️ EASTER EGGS`

```
  ↑ ↑ ↓ ↓ ← → ← → B A    — Konami Code
  Triple-tap logo           — ???
  Three-finger hold         — ???
  Terminal puzzles           — Unlock secrets
  ...and more. Find them.
```

### `📺 CRT MODE`

Toggle chromatic aberration, scanlines, and flicker. Full retro arcade monitor aesthetic.
